
# Hello doc


> Hello doc是一款快译通通过代码生成漂亮的api在线文档工具，支持java快速生成接口文档

运行demo：~~demo~~

## 文档

Link： [详细文档](https://yoqu.gitee.io/hello-doc)


## 特性
* 支持markdown（包含flowchat流程图）
* 参数实体跳转
* java注解支持
* 菜单快速筛选
* 接口搜索
* 接口统计
